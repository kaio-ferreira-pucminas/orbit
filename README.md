## Pasta src  

Este diretório armazena o código-fonte do projeto, que evoluiu de um protótipo front-end para uma **aplicação web full-stack**:

**Front-end**
- HTML5  
- CSS3  
- JavaScript (Vanilla JS, sem frameworks)  

**Back-end** (`src/scripts/backend`)
- Node.js + Express + json-server (API REST)  
- Autenticação JWT, hash de senha (bcrypt), envio de e-mails (Resend)  
- Persistência em `db.json` e uploads de arquivos  

A estrutura contém:  
- Páginas HTML (`src/pages`)  
- Arquivos de estilo (`src/styles`)  
- Scripts JavaScript (`src/scripts`)  
- Back-end e base de dados (`src/scripts/backend`)  
- Recursos visuais (imagens, ícones, etc.)
